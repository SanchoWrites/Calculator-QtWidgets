<h3>Calculator</h3>
<p>Calculator application written in C++ with interface made using Qt widgets.</p>
<p>After calculating a number, said number is saved so you can continue doing operations with it.</p>
<p>You can perform the following operations:</p>
<ul>
  <li>Addition</li>
  <li>Subtraction</li>
  <li>Multiplication</li>
  <li>Division</li>
</ul>
